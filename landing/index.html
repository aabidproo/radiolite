macOS Security Block

If you see a "damaged" or "unverified developer" warning on Mac, follow these steps:

  1. Right-click Radiolite in Applications and select Open.
  2. Or, run this terminal command to bypass the block:
sudo xattr -rd com.apple.quarantine /Applications/Radiolite.app
